# ZERO
Una plataforma B2C de productos ópticos - Empezando desde cero

## Nuestra Historia
Durante más de 25 años hemos conectado fábricas, marcas y ópticas en toda Latinoamérica. Ahora, como segunda generación, llevamos esa experiencia directamente al consumidor final.

**ZERO** es más que una marca de gafas. Es un nuevo comienzo:
- **Cero** intermediarios
- **Cero** complicaciones
- **Cero** límites para elegir lo que realmente te queda bien

Hecha con el legado de la experiencia, pero pensada para una nueva generación que busca calidad, autenticidad y propósito.
